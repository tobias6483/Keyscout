# KeyScout v0.1 Release Notes Draft

KeyScout v0.1 has not been released yet. This draft should be updated with real
QA results before publishing.

## Highlights

- Native macOS menu bar app scaffold
- Best-effort frontmost app shortcut scanning through Accessibility APIs
- Accessibility permission status in the menu with a settings shortcut
- Deterministic unused shortcut suggestion
- Shortcut conflict lookup
- JSON export of the latest shortcut catalog
- Local-only behavior with no account, analytics, telemetry, or cloud sync

## Requirements

- macOS 14 or newer

## Known Limitations

- Accessibility scanning is best-effort and currently limited to frontmost app
  menus.
- Global hotkeys registered outside normal app menus are not detected yet.
- Curated shortcut mappings are not included yet.
- The JSON format is not yet a stable external schema.
- Builds are unsigned and not notarized unless release automation changes before
  v0.1.
