# Support

## Getting Help

Use GitHub issues for:

- Bug reports
- Feature requests
- Shortcut detection limitations
- Documentation problems
- Contribution questions

## Before Filing

Please check:

- Existing issues
- [ROADMAP.md](ROADMAP.md)
- [PRIVACY.md](PRIVACY.md)

## Security Issues

Do not open a public issue for security-sensitive reports. Follow [SECURITY.md](SECURITY.md).
